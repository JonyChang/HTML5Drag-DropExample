    • HTML5 Drag&Drop example application

    Proof of concept project to showcase HTML5 Drag&Drop feature

    Project maintained by vgheri Hosted on GitHub Pages — Theme by mattgraham

    This projects demonstrates how to use the new HTML5 Drag&Drop API, Javascript FileAPI, the XMLHTTPRequest 2 object and Asp.Net MVC 3 to build a web application to drag&drop an image into the browser and load it on the server, providing feedback to the user about the upload status.
